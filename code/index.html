Github

Github is the most popular provider of a web-based Git systems. At its core, Git is used for version controlling files and track any changes made to them. While it is mainly used to allow collaborative development of software, it could in practice be used for any type of files.

While working on a project, instead of relying on storing the files on your own computer you can upload them to a repository on Github. A repository is similar to a folder, where all the files for a particular project are stored. When you (or someone else) make changes to a file, the changes are logged on github. Similarly if you want the latest version of a project Github will allow you to always download the latest version.

While this is just the basics of Github, there are a lot more useful features (like branching, forking and pulling). If you want to know more, please visit one of the links below.
